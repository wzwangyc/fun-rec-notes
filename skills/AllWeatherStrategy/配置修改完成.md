## 🚀 三种使用方式

### 方式 1：直接修改代码（最简单）

编辑 `run_configurable.py` 顶部的配置：

```python
# ========== 在这里修改你的配置 ==========

MY_ETF_LIST = [
    "515450",  # 易方达沪深 300ETF
    "159545",  # 华夏中证 500ETF
    # 添加或删除 ETF...
]

MY_TOTAL_AMOUNT = 10000.0  # 修改金额
MY_LOOKBACK_DAYS = 365     # 修改回看天数
AUTO_SEND_EMAIL = True     # 是否自动发送邮件

# =======================================
```

运行：
```bash
py run_configurable.py
```

---

### 方式 2：修改配置文件

编辑 `config.ini`：

```ini
ETF_LIST = [
    515450,  # 沪深 300ETF
    159545,  # 中证 500ETF
]

TOTAL_AMOUNT = 50000  # 改成 5 万元

LOOKBACK_DAYS = 730  # 改成 2 年数据

SEND_EMAIL = True

EMAIL_TO = "your-email@example.com"
```

运行：
```bash
py run_configurable.py
```

---

### 方式 3：使用默认配置（快速测试）
