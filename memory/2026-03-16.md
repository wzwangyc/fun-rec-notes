## 🏆 数据源排名（按优先级）

### 1️⃣ 东方财富 MX_MacroData（首选）
- **得分：** 4.45/5
- **API Key：** em_3flFbHCB73Vlfu6gZfe1jVtD3upCKkaI
- **位置：** `skills/MX_MacroData/`
- **用法：** `cd skills/MX_MacroData; $env:EM_API_KEY="xxx"; py -m scripts.get_data "中国 GDP"`
- **输出：** CSV + TXT（自动保存到 workspace/macro_data/）
- **优势：** 官方数据、CSV 输出、中文文档
- **适用：** 宏观数据批量获取、定时报告

### 2️⃣ akshare（A 股/中国数据）
- **版本：** 1.18.39
- **安装：** `pip install akshare`
- **用法：** `import akshare as ak; ak.macro_china_gdp_yearly()`
- **优势：** 完全免费、无需 API Key、A 股数据全
- **适用：** A 股历史数据、中国宏观指标

### 3️⃣ Trading Economics（快速验证）
- **网址：** https://tradingeconomics.com/
- **优势：** 秒开网页、全球 195+ 国家
- **适用：** 快速查阅、跨国对比

### 4️⃣ yfinance（美股/ETF）
- **版本：** 1.2.0
- **安装：** `pip install yfinance`
- **用法：** `import yfinance as yf; yf.Ticker('AAPL').history()`
- **优势：** 速度极快 (0.4s)、美股数据全
- **注意：** ❌ 不支持宏观数据（GDP/CPI 等）
- **适用：** 美股/ETF/债券利率

## 💡 使用原则

1. **宏观数据（GDP/CPI/PMI）** → 东方财富 MX_MacroData 优先
2. **A 股/中国市场** → akshare 优先
3. **美股/ETF/金融** → yfinance 优先
4. **快速验证** → Trading Economics 网页
5. **批量获取 + 存档** → 东方财富（CSV 输出）
